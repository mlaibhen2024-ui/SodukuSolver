In this assignment, I chose to use the Depth-First Search (DFS) method because I found it more appealing than the Breadth-First Search (BFS) approach. I also selected DFS because I believed it would be easier to implement, as it allows for searching as deeply as possible through each branch while efficiently backtracking when necessary. Additionally, the use of a stack to store data made the process more straightforward. I think that employing the Depth-First approach improved the overall solution, especially when solving puzzles, as it seems to be more reliable compared to the Breadth-First approach.
